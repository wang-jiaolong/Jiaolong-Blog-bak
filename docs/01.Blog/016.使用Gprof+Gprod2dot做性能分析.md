## Install

```
apt-get install python graphviz
```

```
pip install gprof2dot
```

## Usage

在编译时，使用-pg,

```
gcc -pg demo.c -o demo
```

```
gprof  ./demo gmon.out >profile.txt
```

```
gprof ./demo  | gprof2dot -n0 -e0 | dot -Tpng -o output.png
```
