## 基础

## 1. 获得媒体文件的信息

 ```shell
 ffmpeg -i file_name
 ffmpeg -i file_name -hide_banner
 ```

> 使用 `-hide_banner`后缀可隐藏与文件无关的ffmpeg本身信息
